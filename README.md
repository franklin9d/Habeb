# شركة حبيب الساعدي للتجارة العامة

## نظرة عامة
- **الاسم**: شركة حبيب الساعدي للتجارة العامة
- **الهدف**: موقع عربي احترافي لشركة تجارة عامة عراقية متخصصة في بيع الاجهزة الكهربائية والالكترونية والهواتف الذكية بنظام الاقساط
- **التقنيات**: Hono + TypeScript + Tailwind CSS + Cloudflare Pages

## الروابط

- **معاينة مباشرة**: متاح عبر sandbox
